Clean Cycle Down ～新品羽毛を超える美しさ～

2023.08.11

Share

Clean Cycle Down（CCD）は、表参道布団店が独自に開発したリサイクルダウンの最高峰です。使用済み羽毛から最高品質のダウンだけを選別し、新品を超える清潔さと美しさを実現しました。

リサイクルダウンから最高品質を選別

日本国内で回収された使用済み羽毛布団から、厳格な基準で原料を選別します。まず回収した羽毛を解体し、ダウンボールの大きさ、弾力性、色味を一つひとつ確認。全体のわずか数パーセントに当たる最上質のダウンのみをClean Cycle Down用として選び出します。

白州水による洗浄工程

選別されたダウンは、南アルプスの伏流水である白州水で丁寧に洗浄されます。通常の工業用水ではなく、天然のミネラルウォーターを使用することで、ダウンに残留する油脂分や汚れを優しくかつ徹底的に除去。洗浄後のダウンは、まるで雪のような白さと軽さを取り戻します。

JIS基準の2倍の清浄度

Clean Cycle Downの清浄度は、JIS規格が定める基準値の2倍以上を達成しています。これは新品の羽毛と比較しても遜色ない、いや、むしろ上回る清潔さです。独自の多段階洗浄プロセスにより、アレルギーの原因となる微細な汚れまで徹底的に取り除いています。

清浄度の比較

  • JIS基準値: 500mm以上
  • 一般的な新品羽毛: 500〜800mm
  • Clean Cycle Down: 1,000mm以上

サステナブルな循環型ものづくり

良質な羽毛は、適切にケアすれば100年以上使える天然素材です。Clean Cycle Downは、この素材の素晴らしさを最大限に活かし、廃棄されるはずだった羽毛に新たな命を吹き込みます。環境負荷を最小限に抑えながら、最高品質の眠りを提供する。それがClean Cycle Downの使命です。

新品を超える美しさ。それは、素材の本質を見極め、丁寧に磨き上げることで生まれます。
