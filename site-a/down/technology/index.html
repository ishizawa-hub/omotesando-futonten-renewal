02 — Deep Cleansing

JIS規格の2倍洗浄

私たちは白州の羽毛洗浄工場でJIS規格の２倍の美しさになるまでダウンを徹底的に洗浄します。目に見えない細かいゴミや、付着していた石鹸カス等を取り除く事で新品同様のダウンパワーと輝きを取り戻します。清浄度1000mm以上を実現し、国内最高水準の品質を保証します。

清浄度 1000mm 以上 / JIS規格の2倍基準

03 — Quality Assurance

第三者機関による品質検査

羽毛布団は、一般消費者が中に入っているダウンの性能を知ることは出来ないため、一般的に品質の透明性が低い商品と言われています。だからこそ私たちは国内の専門機関に調査を依頼し、ダウンの品質を常時公開しています。品質においてクリーンでいることはものつくりをする者の責任と考えています。

品質データ常時公開 / 国内専門機関による検査

04 — Fujiyoshida Craftsmanship

富士吉田の職人による手仕立て

山梨県富士吉田は、古くから織物の町として知られる繊維産業の聖地です。私たちの羽毛布団は、この地で熟練の職人たちが一枚一枚手作業で仕立てています。ダウンの充填量を正確に計量し、均一に分散させる技術は、長年の経験と繊細な感覚によって支えられています。

一枚一枚の手仕立て / 富士吉田の熟練職人

05 — Clean Cycle Reform

Clean Cycle Reform

長年ご愛用いただいた羽毛布団を、購入時の品質に復元するリフォームサービスです。使用によってへたったダウンを取り出し、白州の天然水で再洗浄。選別工程を経て新しい側生地に充填することで、新品同様のふくらみと保温性を取り戻します。一枚の布団を長く大切に使い続ける、サステナブルな眠りの形です。

購入時の品質に復元 / サステナブルリフォーム
